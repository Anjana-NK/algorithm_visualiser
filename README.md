Algorithm Visualizer  
----------------  
Description  
---------
  
This is a simple web application that visualizes basic algorithms.
It helps users understand how algorithms work by showing each step visually.
The application demonstrates Bubble Sort and Binary Search using animations so that even beginners can easily follow the process  
  
Technologies Used
--
* Python
* Flask
* HTML
* CSS
* JavaScript

How to run ?  
----
Open browser and go to  
https://algorithm-visualiser-jr31.onrender.com
